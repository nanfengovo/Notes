## 数据库docker-compose
docker-compose.yml
```yaml
services:
  mysql:
    image: mysql:8
    container_name: mysql-dev
    restart: unless-stopped  # 除非手动停止，否则始终重启
    environment:
      # 从 .env 文件读取密码，避免明文写在 yaml 里
      MYSQL_ROOT_PASSWORD: ${MYSQL_ROOT_PASSWORD}
      MYSQL_DATABASE: dev_db  # 启动时自动创建一个开发用的数据库
    ports:
      - "3306:3306"
    volumes:
      # 挂载到宿主机当前目录下的 data 目录
      - ./data/mysql:/var/lib/mysql
    networks:
      - shared-db-net

  sqlserver:
    image: mcr.microsoft.com/mssql/server:2022-latest
    container_name: sqlserver-dev
    platform: linux/amd64
    restart: unless-stopped
    environment:
      ACCEPT_EULA: "Y"
      SA_PASSWORD: ${MSSQL_SA_PASSWORD}
      MSSQL_PID: "Developer"  # 指定版本为开发者版
      MSSQL_ENFORCE_PASSWORD_POLICY: "OFF"
    ports:
      - "1433:1433"
    volumes:
      - ./data/sqlserver:/var/opt/mssql
    networks:
      - shared-db-net

networks:
  shared-db-net:
    name: dev_infrastructure_net  # 给网络起一个固定的名字，方便其他项目接入
    driver: bridge
```
.env
```
MYSQL_ROOT_PASSWORD=abc,123 
MSSQL_SA_PASSWORD=Password123!
```

* 都启动：docker compose up -d*
- **只想用 MySQL：** `docker compose up -d mysql`
    
- **只想用 SQL Server：** `docker compose up -d sqlserver`
    
- **想全关掉：** `docker compose stop`
