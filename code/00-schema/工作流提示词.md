# 日常工作流提示词

这份提示词对应当前知识库结构，可直接复制给 Codex / ChatGPT 使用。

## 1. 收资料
```text
ingest 01-inbox/xxx.md，归档并合并到现有知识页
```

## 2. 问问题
```text
基于我的笔记回答：xxx，并标出可沉淀内容
```

## 3. 沉淀回答
```text
把刚才回答的“可沉淀内容”写回对应页面，并记录日志
```

## 4. 每周整理
```text
执行 lint，输出重复页、孤儿页、过时内容和修复建议
```

## 使用建议
- 新资料先放进 `01-inbox/`
- 有价值的问答结论再写回主题页
- 每周至少执行一次 lint，避免知识库继续碎片化
- 主题页尽量维持统一结构：`结论 -> 适用场景 -> 步骤 -> 坑点 -> 参考`
